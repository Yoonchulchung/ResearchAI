# 대시보드 기능 명세

## 목적

사용자가 앱에 진입했을 때 오늘의 정보, 최근 리서치, 뉴스, 시장/날씨 정보, 검색 진입점을 한 화면에서 확인한다.

## 주요 화면

| 경로 | 설명 |
|------|------|
| `/main` | 메인 대시보드 |
| `/main/search` | 검색/리서치 진입 화면 |

## 주요 기능

### 검색 진입

- 사용자는 대시보드에서 리서치 주제를 입력할 수 있다.
- 입력한 주제는 리서치 생성 플로우로 연결된다.
- 모델 또는 검색 옵션이 필요한 경우 기본 설정값을 사용한다.

### 최근 리서치 요약

- 최근 세션 또는 요약 상태를 카드로 표시한다.
- 사용자는 최근 리서치 상세로 이동할 수 있다.

### 뉴스 섹션

- 주요 뉴스 카테고리와 요약을 표시한다.
- 사용자는 뉴스 상세 화면으로 이동할 수 있다.

### 지도/분쟁 정보

- 월드맵 카드에서 세계 뉴스 또는 분쟁 지역을 시각적으로 표시한다.
- 지도는 자동 회전, 드래그, 줌을 지원한다.

### 날씨/캘린더/마켓

- 날씨, 일정, 시장 정보는 각각 독립 카드로 표시한다.
- 외부 데이터가 없거나 오류가 발생하면 카드 단위 fallback을 표시한다.

## 주요 컴포넌트

| 컴포넌트 | 역할 |
|----------|------|
| `SearchPromptCard` | 리서치 주제 입력 |
| `SummaryCard` | 최근 리서치 요약 |
| `NewsSection` | 뉴스 목록/요약 |
| `WorldMapCard` | 지도/분쟁 시각화 |
| `WeatherCard` | 날씨 |
| `CalendarSection` | 캘린더 |
| `MarketCard` | 시장 정보 |

## 예외/빈 상태

- 뉴스 데이터가 없으면 빈 뉴스 상태를 표시한다.
- 외부 API 오류는 전체 대시보드를 막지 않고 해당 카드만 오류 처리한다.
- 최근 리서치가 없으면 새 리서치를 시작하는 행동을 제공한다.
