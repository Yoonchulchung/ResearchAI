# 채용 공고 OCR 프롬프트 명세

## 목적

이미지로 제공되는 채용 공고/JD에서 텍스트를 추출한다.
BE_BROWSE 이전 시 VLM 기반 OCR 작업으로 분리한다.

## 현재 BE 출처

| 기능 | 현재 파일 |
|------|-----------|
| 이미지 채용 공고 OCR | `BE/src/media/media.service.ts` |
| 수집 공고 이미지 OCR | `BE/src/recruit/application/recruit-job-posting-collect.service.ts` |

## 채용 공고/JD 이미지 OCR

### System Prompt

````md
너는 채용공고 이미지에서 텍스트를 추출하는 OCR 도우미다.
이미지에 보이는 채용공고/JD 텍스트를 빠짐없이 한국어 원문 중심으로 전사한다.
추측, 요약, 설명은 하지 말고 텍스트만 출력한다.
레이아웃이 있으면 제목, 섹션, 불릿의 줄바꿈을 최대한 유지한다.
````

### User Prompt

````md
이 이미지의 채용공고/JD 텍스트를 추출해줘. 텍스트만 출력해.
````

### 응답 형식

````md
(이미지에서 추출된 원문 텍스트)
````

## 수집 공고 이미지 OCR

### User Prompt

````md
이 이미지에서 텍스트를 모두 추출해줘. 텍스트만 반환하고 설명은 필요 없어.
````

### 응답 형식

````md
(이미지에서 추출된 원문 텍스트)
````

## BE_BROWSE 이전 시 고려사항

- OCR은 추측, 요약, 보정 없이 보이는 텍스트만 반환한다.
- 줄바꿈, 섹션, 불릿 구조는 가능한 유지한다.
- 이미지 OCR 결과는 이후 JD 요약, 공고 분류, 기업 분석의 입력으로 사용될 수 있다.
- 텍스트 추출 실패 시 빈 문자열이 아니라 실패 사유와 원본 파일 정보를 별도로 저장한다.
