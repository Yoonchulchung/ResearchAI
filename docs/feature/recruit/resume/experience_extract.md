# 경험 항목 추출 프롬프트 명세

## 목적

자기소개서 또는 문서에서 번호로 구분된 항목을 경험 목록으로 분리한다.
이 결과는 이력서 경험 관리, 자기소개서 평가, 작성 보조의 참고 데이터로 사용할 수 있다.

## 현재 BE 출처

| 기능 | 현재 파일 |
|------|-----------|
| 자기소개서/문서 경험 항목 추출 | `BE/src/recruit/application/documents/documents.service.ts` |

## User Prompt

````md
다음 자기소개서/문서에서 번호로 구분된 각 항목을 분리하여 경험 목록으로 추출해주세요.

문서 내용:
{content}

각 번호별 항목을 분석하여 다음 형식으로 반환하세요:
- title: 항목의 질문/주제를 간결하게 요약 (예: "한화엔진 지원 동기 및 목표")
- content: 해당 항목의 답변 내용 전체

반드시 아래 JSON만 반환하세요 (마크다운 코드블록 없이 순수 JSON):
{"experiences": [{"title": "제목", "content": "내용"}, ...]}
````

## 응답 형식

```json
{
  "experiences": [
    {
      "title": "한화엔진 지원 동기 및 목표",
      "content": "해당 항목의 답변 내용 전체"
    }
  ]
}
```

## BE_BROWSE 이전 시 고려사항

- JSON만 반환하는 작업으로 분리하고, 마크다운 코드블록은 허용하지 않는다.
- `title`은 문항/주제 요약으로 사용하고, `content`는 원문 답변을 최대한 보존한다.
- 분리 실패 시 문서 전체를 하나의 경험 항목으로 저장할지, 실패 상태로 둘지 정책을 정해야 한다.
